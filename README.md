# 🎮✊📄✂️ Rock Paper Scissors PY 2

## 📌 Overview  
This repository contains a **Rock Paper Scissors game built using Python programming language**.  
It is an upgraded version of the classic game where the player competes against the computer in a simple and interactive way.

---

## 🎯 Purpose  
- Practice Python programming fundamentals  
- Improve logic building and decision-making skills  
- Learn randomization and game development basics  
- Build an interactive console-based game  
- Understand user input handling and control flow  

---

## 🧠 Key Features  
- 🤖 Computer vs Player Mode  
- ✊ Rock / 📄 Paper / ✂️ Scissors Choices  
- 🏆 Score Tracking System  
- 🔁 Replay Option  
- ⚡ Random Computer Selection  
- 🖥️ Console-Based Gameplay  

---

## 🛠️ Tech Stack  
- 🐍 Python  
- 🎲 Random Library

- 🧠 Basic Game Logic
